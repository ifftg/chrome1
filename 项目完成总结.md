## 🚀 使用指南

### 1. Chrome插件安装
```bash
1. 打开 Chrome 浏览器
2. 访问 chrome://extensions/
3. 开启"开发者模式"开关
4. 点击"加载已解压的扩展程序"
5. 选择 yifu-youtube-downloader 文件夹
6. 安装完成，开始使用！
```

### 2. 网站本地预览
```bash
# 进入网站目录
cd yifu-website

# 启动本地服务器
python -m http.server 8000
# 或者
npx serve .

# 访问 http://localhost:8000
```

### 3. 项目打包
```bash
# Windows用户
双击运行 build.bat

# 将生成：
# dist/yifu-youtube-downloader.zip  (插件安装包)
# dist/yifu-website.zip             (网站部署包)
```
